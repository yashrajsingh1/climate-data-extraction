# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-03-26

### Added

- Flask web dashboard for one-click and batch climate extraction
- CLI interface for scripted single-company and batch workflows
- Multi-source report discovery (Google, DuckDuckGo, optional Playwright)
- Hybrid emissions extraction with optional LLM fallback
- SQLite data models and repository layer
- Unit tests and GitHub Actions CI workflow
- Deployment and operational documentation
